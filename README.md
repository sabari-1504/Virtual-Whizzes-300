## Overview
This project is an interactive learning project designed to support children through calm, friendly, and engaging experiences.  
The project includes two builds:

1. A **Desktop AI Conversation Build (EXE)** focused on voice-based interaction with a friendly NPC.
2. An **Android VR Build (APK)** that provides immersive learning scenes covering basic concepts through exploration and interaction.

Both builds are designed to be simple, predictable, and child-friendly.

---

## Build 1: Convai Desktop Build (EXE)

### Description
The Convai Desktop Build provides a conversational experience where children interact with a friendly AI character using voice.  
The NPC responds in a calm, stable, and supportive manner, making the interaction feel like a friendly chat rather than a lesson.

### How It Works
1. The user speaks into a microphone.
2. Speech is converted into text.
3. The AI processes the input and generates a friendly response.
4. The response is spoken back using text-to-speech.
5. The conversation remains short, predictable, and easy to follow.

### How to Run
1. Open the desktop build folder.
2. Double-click the `.exe` file.
3. Make sure a microphone and speakers/headphones are connected.
4. Start speaking to interact with the AI character.

---

## Build 2: Android VR Build (APK)

### Description
The Android VR Build is an immersive virtual reality learning experience.  
It allows children to explore different learning scenes in a structured and visually engaging environment using a VR headset.

### How It Works
1. The user launches the app on an Android VR device.
2. The user navigates through different learning scenes.
3. Each scene focuses on a specific educational concept.
4. Visual interaction and repetition help reinforce learning.
5. The experience is calm, guided, and easy to understand.

---

## VR Scenes Included

### 1. Familiarization Scene
This scene helps users get comfortable with the VR environment.  
Children learn how to look around, move, and interact with objects.  
It serves as an introduction to VR controls and navigation.

### 2. Alphabet Learning Scene
This scene introduces letters of the alphabet using clear visuals.  
Children interact with letters to improve recognition.  
It supports early literacy skills in a simple way.

### 3. Numbers Learning Scene
Children learn basic numbers and counting through visual interaction.  
Numbers are displayed clearly and repeatedly.  
This scene helps build early numerical understanding.

### 4. Chocolate Buying Scene
A simple role-play scenario where children experience buying an item.  
It introduces concepts like selection and quantity.  
The scene helps build basic real-world interaction skills.

### 5. Gestures Learning Scene
This scene demonstrates common gestures visually.  
Children observe and learn body movements and expressions.  
It supports gesture recognition and motor awareness.

### 6. Body Parts Learning Scene
Children learn to identify different body parts.  
Body parts are shown clearly with visual guidance.  
This scene supports basic anatomy awareness.

### 7. Colors Learning Scene
Children explore different colors using objects and environments.  
Colors are repeated for easy recognition.  
This scene helps strengthen color identification skills.
